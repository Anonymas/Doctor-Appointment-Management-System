# Doctor Appointment Management System

A web-based system that allows patients to book appointments with doctors while enabling administrators to manage doctors, patients, and schedules efficiently.

## Features
- Patient registration and login
- Doctor management
- Appointment booking system
- Admin dashboard
- Secure authentication
- Responsive UI

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap

## Installation
1. Clone the repository
2. Import the database.sql file into MySQL
3. Configure database connection in the config file
4. Run the project using XAMPP or WAMP
